┌──────────────────────────────────────────────────────────────┐
│  📋 PASSO 1: INSTALAÇÕES (Apenas na primeira vez)           │
└──────────────────────────────────────────────────────────────┘

  1️⃣  Java 17+
      ┌────────────────────────────────────────────────────┐
      │ sudo apt install openjdk-17-jdk                    │
      │ java -version    # Verificar (com "version")      │
      └────────────────────────────────────────────────────┘

  2️⃣  Android Studio + SDK
      ┌────────────────────────────────────────────────────┐
      │ Download: https://developer.android.com/studio     │
      │ Instalar Android Studio                            │
      │ Instalar Android SDK durante setup                 │
      └────────────────────────────────────────────────────┘

  3️⃣  Configurar Variáveis de Ambiente
      ┌────────────────────────────────────────────────────┐
      │ Adicione ao ~/.zshrc ou ~/.bashrc:                 │
      │                                                     │
      │ export ANDROID_HOME=$HOME/Android/Sdk              │
      │ export PATH=$PATH:$ANDROID_HOME/platform-tools     │
      │ export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64│
      │                                                     │
      │ Depois: source ~/.zshrc                            │
      └────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│  🚀 PASSO 2: GERAR APK (Sempre que quiser uma nova versão)  │
└──────────────────────────────────────────────────────────────┘

  ⚡ MÉTODO AUTOMÁTICO (RECOMENDADO):
  ┌────────────────────────────────────────────────────────────┐
  │ cd ~/Documentos/Pessoal/autocare-landing-page/            │
  │    Garagem-inteligente/App-client/app-client               │
  │                                                            │
  │ ./build-android.sh                                         │
  │                                                            │
  │ ✨ Este script faz TUDO automaticamente:                  │
  │   - Build do Vue                                           │
  │   - Sync Capacitor                                         │
  │   - Gera APK                                               │
  │   - Copia para raiz                                        │
  │   - Oferece instalação no celular                          │
  └────────────────────────────────────────────────────────────┘

  📦 APK GERADO: garagem-inteligente-debug.apk

┌──────────────────────────────────────────────────────────────┐
│  📲 PASSO 3: INSTALAR NO CELULAR                            │
└──────────────────────────────────────────────────────────────┘

  ═══════════════════════════════════════════════════════════
    OPÇÃO A: Via USB (Mais Rápido) 🔌
  ═══════════════════════════════════════════════════════════

  1. Habilitar Depuração USB no celular:
     ┌───────────────────────────────────────────────────┐
     │ • Configurações → Sobre o telefone                │
     │ • Toque 7x em "Número da versão"                  │
     │ • Configurações → Opções do desenvolvedor         │
     │ • Ative "Depuração USB"                           │
     └───────────────────────────────────────────────────┘

  2. Conectar celular via USB

  3. Instalar APK:
     ┌───────────────────────────────────────────────────┐
     │ adb devices    # Verificar se detectou            │
     │ adb install -r garagem-inteligente-debug.apk      │
     └───────────────────────────────────────────────────┘

  ═══════════════════════════════════════════════════════════
    OPÇÃO B: Transferir Manualmente 📁
  ═══════════════════════════════════════════════════════════

  1. Copiar o arquivo "garagem-inteligente-debug.apk" 
     para o celular (via USB, Bluetooth, ou Cloud)

  2. No celular:
     ┌───────────────────────────────────────────────────┐
     │ • Abrir gerenciador de arquivos                   │
     │ • Navegar até onde copiou o APK                   │
     │ • Tocar no arquivo APK                            │
     │ • Permitir "Instalar de fontes desconhecidas"     │
     │ • Seguir instruções de instalação                 │
     └───────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│  🐛 PROBLEMAS COMUNS E SOLUÇÕES                              │
└──────────────────────────────────────────────────────────────┘

  ❌ "Command not found: ./build-android.sh"
     ✅ chmod +x build-android.sh

  ❌ "JAVA_HOME not set"
     ✅ export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64

  ❌ "Android SDK not found"
     ✅ export ANDROID_HOME=$HOME/Android/Sdk

  ❌ "Gradle build failed"
     ✅ cd android && ./gradlew clean && cd ..

  ❌ "No device found" (adb)
     ✅ Verificar se USB debugging está ativo
     ✅ Aceitar prompt de autorização no celular
     ✅ Tentar: adb kill-server && adb start-server
